The Range42 backend API is documented via OpenAPI. Browse the live reference:

- Swagger UI: `/docs/swagger` on a running backend
- ReDoc: `/docs/redoc`
- OpenAPI JSON: `/docs/openapi.json`

The API specification lives in [`range42-backend-api`](https://github.com/range42/range42-backend-api).
